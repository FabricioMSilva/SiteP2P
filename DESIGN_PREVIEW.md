## 📊 Componentes & Props

```typescript

<Header />
// Props: Nenhum (conteúdo fixo)
// Output: Slogan + Onda animada

// SERVICE EXPLANATION
<ServiceExplanation />
// Props: Nenhum (conteúdo fixo)
// Output: 3 cards + Benefícios

// CTA
<CTA />
// Props: Nenhum (usa constantes)
// Output: Botão WhatsApp funcional

// FOOTER
<Footer />
// Props: Nenhum (ano automático)
// Output: 3 colunas + Copyright
```

## 🌊 Animações Implementadas

1. **Slide-Up**
   ```
   Elemento aparece subindo da posição inicial
   Usado em: Header, Cards
   Tempo: 0.6s
   ```

2. **Hover Effects**
   ```
   Botão no hover:
   - Scale up (110%)
   - Sombra aumenta
   - Cor mantém
   ```

3. **Shadow Effects**
   ```
   Cards no hover:
   - Sombra XL
   - Transição suave
   ```

## 📱 Breakpoints Tailwind

| Tamanho | Prefixo | Largura |
|---------|---------|---------|
| Mobile | - | < 640px |
| Tablet | `md:` | 640-1024px |
| Desktop | `lg:` | > 1024px |

Exemplos:
- `text-2xl md:text-4xl` - Tamanho texto varia
- `grid-cols-1 md:grid-cols-3` - Colunas variam
- `p-4 md:p-8` - Padding varia
